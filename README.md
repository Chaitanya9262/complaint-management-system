# Complaint Management System (CMS)

## Project Overview

A web-based Complaint Management System developed using Spring Boot, React.js, and MySQL. The system allows users to register, submit complaints, track complaint status, and enables administrators to manage complaints efficiently.

## Features

* User Registration & Login
* JWT Authentication
* Role Based Access Control (Admin/User)
* Complaint Submission
* Complaint Tracking
* Complaint Status Updates
* Reports & Analytics
* Profile Management
* Admin Dashboard
* User Dashboard

## Technologies Used

### Frontend

* React.js
* Bootstrap
* Axios
* Recharts

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* Maven

### Database

* MySQL
